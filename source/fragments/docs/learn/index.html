  • Step 8.

    Create A Project

    To create a new Trio project, run the following from the command line:

    trio n|new path/to/project

    where "path/to/project" is the path to where you want to create the new project.

  • Step 13.

    Build A Project

    To build a project for development, from the project's root folder run the following from the command line:

    trio b|build
  • Step 14.

    Release A Project

    To build a project for release, from the project's root folder run the following from the command line:

    trio r|release
  • Step 15.

    Run A Project

    To run a project, from the project's root folder run the following from the command line:

    trio s|serve
