JSON Data

At time of build, Trio creates the dataCatalog metadata object from the individual JSON files it finds in folder root/source/data and exposes the dataCatalog to tag-based callbacks.

As an example, the file root/source/data/personalization.json would be exposed to tag-based callbacks as:

  • file location: root/source/data
  • file type/content: .json
