Tag Attributes

data-trio-* tag attributes are used as markers to indicate that a specific action is associated with its HTML tag.

  • data-trio-fragment: Applicable in template assets and is used to indicate that the HTML tag is the target of a fragment asset's content
  • data-trio-include="name of include": Must include a file extension (e.g. .html | .md). Applicable in template and fragment assets and is used to indicate that the HTML tag is the target of the named include asset's content.
  • data-trio-callback="name of tag-based callback": File extension is not required; it is assumed that it is .js. Applicable in template, fragment and include assets and is used to indicate that the HTML tag is requesting that Trio call the named tag-based callback.
  • data-trio-link: Applicable when your Web hosting services hosts your site in a subfolder and requires that your URLs be prepended with a base URL. See Base URL in Page Generation for more information.
