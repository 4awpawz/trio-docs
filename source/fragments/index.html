Extend Your Composites Using Tag-Based JavaScript (ES6+) Callbacks!

Trio doesn't require a templating engine or a front-end framework for extending your composites. Instead, Trio exposes its tag-based callback mechanism.

Include the data-trio-callback attribute in any HTML tag and Trio will call the JavaScript module that it names, passing it the tag, and the metadata that Trio has collected.

Fragment asset source/fragments/products.html:

<!--
template: products.html
title: Products By Category
appendToTarget: true
data: productsByCategory.json
-->
<h1 class="product-list-title">Product List By Category</h1>
<ul data-trio-callback="listAllProductsByCategory" class="product-list">
    <li class="product-in-stock product-in-stock-heading">
        <span>Category</span>
        <span>Name</span>
        <span>Price <span class="product-in-stock-heading--smaller">(US$)</span></span>
    </li>
</ul>

Template asset source/templates/products.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <link rel="stylesheet" href="/css/main.css">
</head>
<body>
    <main>
        <article data-trio-fragment></article>
    </main>
</body>
</html>

        "category": "fruit",

Tag-Based JavaScript asset source/callbacks/listAllProductsByCategory.js:

module.exports = ({$tag, site}) =>
    site.dataCatalog.productsByCategory
        .filter(prod => prod.inStock)
        .forEach(prod => $tag.append(`
            <li class="product-in-stock">
                <span>${prod.category}</span>
                <span>${prod.name}</span>
                <span>${prod.price.toFixed(2)}</span>
            </li>
        `));

Trio creates a composite from the above template and fragment assets. It then calls the tag-based callback declared in the fragment asset. The tag-based callback then extends the composite using the data from the productsByCategory.json file to produce the "Product List By Category" report.

Create Modern-Day Blogs With The Features Your Readers Have Come To Expect!

Trio was built from the ground up to be used as a blogging platform.

So features, such as article categories and tags, category, tag, and archive pages, article page navigation, are a snap to implement using Trio's structured blog-specific metadata and its tag-based callback mechanism.
